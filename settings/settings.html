Doggify Settings


Breed:
Continous Doggos



Breed allows you to limit what dog pictures are used to a specific breed. Selecting "All doggos" will allow the extension to select a random dog picture from any breed.

If "Continous Doggos" is ticked then the extension will automatically replace all images on a web page with dogs.

The option to right click an image and select "Doggify me!", which will replace the image with a random dog, is always enabled.
